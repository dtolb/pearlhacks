## Pre-reqs
* [Bandwidth Account](http://app.bandwidth.com)
* [Node Installed](https://nodejs.org/en/)

## Start & Run
1. Clone this repo
2. Install the modules
2.1 `npm install`
3. Run the app
3.1 `npm start`
4. Visit the site
4.1 `localhost:3000`
